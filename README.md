# Task Management System

## Features
- User Registration & Login
- Create, Update, Delete Tasks
- Assign tasks to users

## Tech Stack
- Spring Boot
- MySQL
- JPA

# Task_management_System
Developed a Task Management System using Spring Boot, enabling users to perform CRUD operations on tasks with user-based task assignment. Implemented REST APIs following layered architecture principles and used Spring Data JPA for efficient database interaction with MySQL. Ensured clean code practices and handled exceptions effectively.
